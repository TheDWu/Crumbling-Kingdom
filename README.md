## Controls
**Movement**
Arrow Keys to move
Double tap Up Arrow to double jump
**Attack**
X - Basic Attack
Z - Ability 1
C - Ability 2 

Abilities will be throughout the game
**Interaction**
V - Open Chest 
ESC - Pause the game or exit to menu 

To have full access to the game, send a DM.
